## Lists

### Ordered Lists

````md
1. Fist item
2. Second item
    1. Ordered sub list item must have four spaces
3. Third item
    * Unordered sub list item
4. etc.
````

1. First item
2. Second item
    1. Ordered sub list item must have four spaces
3. Third item
    * Unordered sub list item
4. etc.

### Unordered Lists

````md
* Unordered list can use asterisks
- Or minuses
+ Or pluses
````

* Unordered list can use asterisks
- Or minuses
+ Or pluses
